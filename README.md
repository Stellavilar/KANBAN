# KANBAN

## Kanban created with pure Javascript

This JS application is a kanban like "Trello" created with Vanilla Javascript and connected to a PostgreSQL database.

Here are the technologies used:
- NodeJS
- ExpressJS
- PostgreSQL
- Sequelize
- Rest API
- Insomnia
- Sanitizer
- Multer
- Browserify
- SortableJS
